# Security policy

## Supported versions

This project has no released versions yet.

## Secrets

Never commit:

- Kavita API keys.
- Server URLs that identify a private home server.
- Test files containing private books.
- Keystores or signing keys.

## Reporting

Until the project has a public security process, report security issues privately to the repository owner.

## DRM

This project does not support DRM removal or DRM bypass. Reports or pull requests adding such behavior should be rejected.
